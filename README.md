# Oracle Q&A Source

A simple Question and Answer source for Oracle SQL and PL/SQL, built with **Vue 3 (Options API)**, **jQuery AJAX**, and **Tailwind CSS**.  
This project is hosted as a static site on **GitHub Pages**.
